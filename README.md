To run in your local machine:
Firstly clone the project using:
git clone https://github.com/kananarora1/Task-Manager-in-MERN.git
then:
cd Task-Manager
After this, if you don't have docker or docker-compose installed,
install it from here:
https://www.docker.com/
After that run this command in your terminal:
docker-compose up --build
Then switch to the localhost port on which it is running.
